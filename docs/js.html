You can call your Joined Node Tasks easily with a single HTTP call, or using other libraries to send the request, but we've also made it easy to use it with your JavaScript apps using our library.

Use the Joined Node JavaScript Library to call your Joined Node Tasks from the browser using a require-like pattern. Run node.js code without a backend.

code sample 

var joinednode = require('joinednode')('<your-joinednode-container>');
joinednode('<your-joinednode-app>').then(function(result) {
	console.log(result);
});

Install

There are two ways you can run the Joined Node Javascript Library, you can run it from a browser or from another node.js script.

Let's look at running it from a browser first.

Inside a browser

You want to use our CDN to include the library:

code sample 

<script src="https://cdn.joinednode.com/joinednode.min.js"></script>

If you haven't created a Joined Node account yet, you can create one now

Once created, every task is callable by using your unique container id and the task's unique id. This is how you call it:

code sample 

var joinednode = new joinednode('<your-joinednode-container>');
joinednode('<your-joinednode-app>').then(function(result) {
	console.log(result);
});

Install via NPM:

code sample 

npm install joinednode

If you haven't created a Joined Node account yet, you can create one now

Once created, every task is callable by using your unique container id and the task's unique id. This is how you call it:

code sample 

var joinednode = require('joinednode')('<your-joinednode-container>');
joinednode('<your-joinednode-app>').then(function(result) {

Sending parameters

code sample 

var joinednode = require('joinednode-require')('<your-joinednode-container>');
joinednode('<your-joinednode-app>', {foo: 'bar'}).then(function(result) {
	console.log(result);
});

Specifying an HTTP method

code sample 

joinednode.get('<your-joinednode-app>');
joinednode.post('<your-joinednode-app>', { foo: 'bar' });

Calling private tasks

Save your task as Private via the Joined Node Dashboard.

Then create a token via our API:
code sample 

$ curl -X POST https://api.joinednode.com/create-token/<your-joinednode-container>

you then call your task using a token obtained from Joined Node:

code sample 

var joinednode = require('joinednode')('<your-joinednode-container>');
joinednode.withAuth(token)('<your-joinednode-app>').then(function(result) {
	console.log(result);
});
