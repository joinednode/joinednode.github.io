When you create a Joined Node Task, there is a separate text box where you can enter JSON formatted variables, these are then based to your context object as an array inside context.env and are treated as secret parameters. These parameters are stored encrypted when not being executed.

secret.json 

{
	"author": "Roger"
}
hello.js 

module.exports = function(context, callback) { 
	callback(null, {message: "Hello " + context.data.who + ', sincerely ' + context.env.author + '!' }); 
}

These variables can be useful for storing information such as mongodb urls, twilio information, etc, rather than passing them each time you make the call.
