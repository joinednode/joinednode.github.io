You can pass parameters to your code using a query string such as ?who=myself. All URL query parameters will be passed to the recipe when it runs.

hello.js 

module.exports = function(context, callback) { 
	callback(null, {message: "Good morning " + context.data.who + '!' }); 
}

Passing ?who=me in your URL will add a parameter called who to your context.data object and will be accessible to your recipe.

context.data will also work on POST requests.
