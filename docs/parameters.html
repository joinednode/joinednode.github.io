You can pass parameters to your code using URL query string of the request. All URL query parameters will be propagated to the recipe when it runs.

To take advantage of this feature, define your JavaScript function such that it accepts two arguments instead of one: the context and the callback:

hello.js 

module.exports = function(context, callback) { 
	callback(null, {message: "Good morning " + context.data.who + '!' }); 
}

Passing ?who=me in your URL will add a variable called who to your context.data object and will be accessible to your Recipe.

All allowed URL query parameters of the request will be provided to the code in the form of context.data JavaScript object.

context.data will also work on POST requests.
