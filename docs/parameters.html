You can pass paramaters to your code using URL query string of the request. All URL query parameters will be propagated to the task when it runs.

To take advantage of this feature, define your JavaScript function such that it accepts two arguments instead of one: the context and the callback:

hello.js 

module.exports = function(context, callback) { 
	callback(null, {message: "Hello " + context.data.who + '!' }); 
}

All allowed URL query parameters of the request will be provided to the code in the form of context.data JavaScript object.
