Getting Started

Get up and running in 30 seconds or less

Getting started with Joined Node is actually really simple, you just Sign up, and create your first Task.

Tasks are the heart of Joined Node, a Task is a snippet of code that can be called using a simple HTTP request, either directly in a browser or anywhere else. These Tasks can be very simple, or they can be extended as much as you wish.

Creating your first task

Once you've signed up, let's create a task called hello world:

hello.js 

module.exports = function(context, callback) { 
	callback(null, {message: "Hello " + context.data.who + '!' }); 
}

This will create a unique URL, which is based on your unique container and task, so you would see a URL like this:

https://api.joinednode.com/run/black-surf-17638/polished-firefly-55389

Then, you would call it with: https://api.joinednode.com/run/black-surf-17638/polished-firefly-55389?who=me via curl, or from a web request, which would return a JSON result with message that says hello to the name you passed as who.

Joined Node Tasks use a slightly different programming model that traditional node.js scripts do, mostly this is designed to help make creating your tasks quicker, while also giving you flexibility.
